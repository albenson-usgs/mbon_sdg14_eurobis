Marine Biodiversity Observation Network Product Development for the Sustainable Development Goal 14 using EurOBIS data
======================================================================================================================

-----------
Contact:
-----------
Abby Benson (albenson@usgs.gov)


-----------
Purpose:
-----------
This repository contains R code for analyzing data from the European Ocean Biogeographic Information System for the Norwegian Sea developing a product in association with the Sanctuaries Marine Biodiversity Observation Network for the GEO BON plenary in October 2017 for the Sustainable Development Goal 14
